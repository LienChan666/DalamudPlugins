# Dalamud 插件发布仓库

本仓库仅用于存放个人 Dalamud 插件的发布产物，不在这里维护插件源码。

AutoRetainer 简体中文汉化源码位于 [LienChan666/AutoRetainer](https://github.com/LienChan666/AutoRetainer)：

- `master`：当前维护和发布分支。
- `deprecated`：历史分支，仅供查看，不再构建或发布。

Dalamud 自定义插件仓库地址：

```text
https://raw.githubusercontent.com/LienChan666/DalamudPlugins/master/pluginmaster.json
```
